## What is ClearPath?

ClearPath is a browser extension that transforms any webpage into something a person can actually read, understand, and engage with. It started from a simple observation: tools like AAC apps help people communicate, but the web itself still isn't built for many of those people.

ClearPath bridges that gap — directly in the browser, on any site, with no account required.

No data collected. No server. No login. Everything stays in your browser.

## Who This Is For

ClearPath is built with and for:

- **AAC users** who need web content simplified or read aloud
- **People with dyslexia** who need better typography and reading support
- **People with cognitive disabilities** who need plain language and symbols
- **Speech-Language Pathologists (SLPs)** who want to share accessible settings with clients
- **Caregivers and educators** who support people with communication needs
- **Anyone** who just wants the web to be a little easier to navigate

---

## Built by the Team Behind [Sprout](https://sproutaac.org)

ClearPath is created and maintained by the same people building Sprout, an AAC communication app. The philosophy is the same: remove barriers between people and the information they need.

## Get Involved

ClearPath is community-driven. There are meaningful ways to contribute regardless of your background:

- **Developers** — build features, fix bugs, improve performance
- **Designers** — improve the UI, create icons, work on the symbol library
- **SLPs & AAC specialists** — advise on symbol accuracy, test with users, write documentation
- **Writers** — improve our plain-language docs, translate into other languages
- **Everyone** — report bugs, suggest features, share with people who'd benefit

Start with the [Contributing Guide](https://github.com/clearpath-ext/clearpath-extension/blob/main/CONTRIBUTING.md) or browse [`good first issue`](https://github.com/clearpath-ext/clearpath-extension/labels/good%20first%20issue) labels.
